# ECIOT

#### 介绍
IOT物联网基础平台系统，版本号：eciotv1.0
ECIOT 是目前国内为数不多能覆盖有线设备、无线物联网设备接入等物联网云服务功能的智能操作系统，可以部署在边缘侧和云端同时为万物互联提供可靠安全稳定的终端接入、协议适配、智能策略、数据存储、图表分析、三维可视化和应用使能等核心功能。
